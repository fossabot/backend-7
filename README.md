## Description
[![FOSSA Status](https://app.fossa.com/api/projects/git%2Bgithub.com%2FFridgy-app%2Fbackend.svg?type=shield)](https://app.fossa.com/projects/git%2Bgithub.com%2FFridgy-app%2Fbackend?ref=badge_shield)


NestJS backend for Fridgy app

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
